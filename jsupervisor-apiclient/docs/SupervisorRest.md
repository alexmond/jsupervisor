# SupervisorRest

Supervisor system information including hardware and OS details

## Properties

| Name                | Type        | Description                                   | Notes      |
|---------------------|-------------|-----------------------------------------------|------------|
| **nodeName**        | **String**  | The name of the node in the supervisor system | [optional] |
| **description**     | **String**  | Description of the node                       | [optional] |
| **physicalMemory**  | **String**  | Total physical memory available in the system | [optional] |
| **physicalCpu**     | **Integer** | Number of physical CPU cores                  | [optional] |
| **operatingSystem** | **String**  | Operating system information                  | [optional] |
| **availableMemory** | **String**  | Currently available memory in the system      | [optional] |
